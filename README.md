# Projet : Plateforme de Freelances

## Nom et classe
- Nom : Zakariya Sakho
- Classe : L1 DS

## Description
Ce projet est une plateforme web permettant aux utilisateurs de rechercher et de découvrir des freelances dans différents domaines. Les visiteurs peuvent consulter les profils, explorer les catégories de services et lire les témoignages. Le site est conçu avec une interface moderne, responsive et facile à utiliser.

## Technologies utilisées
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Fonctionnalités principales
- Page d'accueil avec section Hero et présentation des services.
- Catalogue de freelances avec cartes Bootstrap et filtres par catégorie.
- Section « Comment ça marche » en Bento Grid.
- Catégories de services avec effets Hover CSS.
- Carousel Bootstrap pour les témoignages.
- Interface responsive adaptée aux ordinateurs, tablettes et smartphones.
